The file meta_schema.xsd contains the XML Schema meta-schema, or schema for
schema documents, from the W3C Recommendation at:

    XML Schema Part 1: Structures Second Edition
    W3C Recommendation 28 October 2004
    https://www.w3.org/TR/xmlschema-1/#normative-schemaSchema

The file datatypes.xsd contains the definition of simpleType element and all of
its members, from the W3C Recommendation at:

    XML Schema Part 2: Datatypes Second Edition
    W3C Recommendation 28 October 2004
    https://www.w3.org/TR/xmlschema-2/#schema
